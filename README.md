# Aplikasi-KalenderKu

Aplikasi pengelola jadwal harian dengan desain modern berbasis **Glassmorphism**. Proyek ini mengedepankan estetika visual yang bersih dan pengalaman pengguna yang intuitif.

## Fitur Antarmuka (UI)

Berikut adalah rincian tampilan utama dari Aplikasi-KalenderKu:

### 1. Beranda & Jadwal Harian
Halaman utama yang memberikan ringkasan agenda pengguna secara cepat.
- **Header Informasi Waktu:** Menampilkan hari dan tanggal secara menonjol.
- **Weekly View:** Navigasi kalender ringkas berbasis minggu dengan sorotan (*outline* merah) pada hari aktif.
- **Empty State:** Tampilan kartu jadwal yang bersih saat tidak ada agenda.
- **Floating Action Button (FAB):** Tombol "+" dengan gradien warna yang menarik untuk akses cepat tambah jadwal.

### 2. Pemilihan Tanggal (Calendar View)
Antarmuka kalender penuh untuk navigasi antar bulan.
- **Desain Glassmorphism:** Kartu kalender semi-transparan di atas latar belakang sinematik.
- **Indikator Aktif:** Penanda lingkaran berwarna *cyan* untuk memperjelas tanggal yang dipilih.
- **Navigasi Intuitif:** Tombol panah untuk perpindahan bulan yang mudah.
- **Microcopy:** Instruksi *subtle* "Ketuk tanggal untuk melihat jadwal" untuk memandu pengguna.

### 3. Profil Pengguna
Halaman personalisasi identitas pengguna dalam aplikasi.
- **Identitas Digital:** Menampilkan foto profil, nama User , dan status User.
- **Customization:** Fitur "Ganti Wallpaper Utama" yang memungkinkan pengguna mengubah estetika latar belakang aplikasi secara instan.
- **Navigasi Konsisten:** *Bottom Navigation Bar* yang seragam di seluruh halaman untuk kemudahan perpindahan menu.
